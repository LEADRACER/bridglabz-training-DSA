# Bridglabz Training - DSA

This repository contains implementations of various Data Structures and Algorithms (DSA) problems as part of the Bridglabz training.

## Structure

- **[stacks-and-queues/](file:///root/.gemini/antigravity/scratch/bridglabz-training-DSA/stacks-and-queues)**:
    - `code1.java`: Queue implementation using two Stacks.
    - `code2.java`: Sorting a Stack using recursion.
    - `code3.java`: Stock Span Problem solution.
    - `code4.java`: Sliding Window Maximum using a Deque.
    - `code5.java`: Circular Tour (Gas Station) Problem.

- **[hash-map-and-hash-function/](file:///root/.gemini/antigravity/scratch/bridglabz-training-DSA/hash-map-and-hash-function)**: 
    - (Upcoming implementations for Hash Tables and Hash Functions)
